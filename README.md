# Hokey Retrospective

A dynamic React-based website showcasing the Ice Warriors hockey team from Bucharest. This project presents team information, player profiles, standings, news, sponsors, and official merchandise in an interactive single-page application.

## Features

- **Team Overview**: Display team history, stats, and facilities
- **Player Profiles**: Detailed cards for each team member with stats
- **League Standings**: Current position in the championship
- **Latest News**: Updates on transfers and team events
- **Sponsors Section**: Showcase of official partners
- **Official Store**: Team merchandise with pricing

## Technologies Used

- React 17
- Babel (for JSX transpilation)
- CSS for styling
- Hosted on GitHub Pages

## Live Demo

View the live website at: [https://itsiamdev.github.io/Hokey-Retrospective/](https://itsiamdev.github.io/Hokey-Retrospective/)

## Getting Started

To run this project locally:

1. Clone the repository
2. Open `index.html` in a web browser
3. No build process required - it's a static site using CDN links

## Project Structure

- `index.html` - Main HTML file
- `App.js` - React application logic
- `styles.css` - Styling for the website
- `ice-hockey-puck.png` - Favicon
